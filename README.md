# PasswordWallet

Password management app.  
Check the progress at https://github.com/raxell/PasswordWallet/projects/1

## Usage

To use the app:

* Install the required dependencies
```
npm install
```

* Build CSS, JS files
```
npm run build
```

* Start the server
```
npm start
```

Go to `http://localhost:3000/` to view the app.
